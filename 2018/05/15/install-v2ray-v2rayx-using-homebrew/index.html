v2ray及V2RayX是啥？

V2Ray 是一个模块化的代理软件包，它的目标是提供常用的代理软件模块，简化网络代理软件的开发。

简单说 v2ray 就是翻墙代理软件（但不止于软件，是一个平台）。V2RayX 就是 macOS 下一个简单的 v2ray 的GUI程序。
